# LetsMakeMoney

[简体中文](README.md)

LetsMakeMoney is a Windows desktop pet and earnings-progress widget. The current release baseline is v0.6 Beta. v0.7 is private development focused on open-source readiness and is not a published release.

> The complete bilingual README is scheduled for V07-E1. This file currently establishes the English license boundary required by V07-A1.
