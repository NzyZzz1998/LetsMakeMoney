# 参与者行为准则

本项目采用 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 的核心原则：尊重不同背景与经验，以建设性的方式讨论问题，不骚扰、不歧视、不公开他人的私人信息。

## 适用范围

本准则适用于仓库 Issue、Pull Request、Discussion 和代表项目参与的其他公开空间。

## 执行

请通过 GitHub Private Vulnerability Reporting 或仓库维护者的非公开渠道报告行为问题，不要在公开 Issue 中披露敏感信息。维护者可删除不当内容、限制参与或永久禁止严重违规者。
