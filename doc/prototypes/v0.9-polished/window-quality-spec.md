# LetsMakeMoney Windows 窗口质感合同 R1

## 定位

- 平台：Windows 桌面小工具。
- 主要任务：扫一眼收入与工作状态，必要时快速调整配置。
- 视觉方向：中性桌面底、接近白纸的单层窗口、深咖啡文字、金币黄动作、鼠尾草绿状态。
- 当前阶段判断：已有稳定品牌色和控件方向，但窗口结构、密度和层级尚未形成统一合同。

## 本轮必须解决

### P0

1. Panel 展开态必须完整保留四个圆角，内容不能把宿主窗口撑出或裁掉底边。
2. Settings 与 Wizard 只能有一个视觉壳体，禁止“宿主窗口 + 奶油外框 + 白色内容卡”的套壳结构。
3. 折叠 Panel 只保留今日已赚、当前状态和工作进度，不能塞入月累计、时薪、完整安排。

### P1

1. 所有窗口使用同一套标题、分割线、按钮、输入、开关、滑块和反馈语言。
2. 内容分组依靠对齐、留白和分割线，设置行不再做成独立卡片。
3. 金币黄只用于主动作、当前选择和进度；绿色只用于正常/启用状态。
4. 窗口阴影承担浮层关系，外框不再同时使用宽阴影和明显边线。

## Token

| 角色 | 值 | 用途 |
| --- | --- | --- |
| 桌面底 | `#E7E8E5` | 原型舞台与窗口外背景 |
| 窗口纸面 | `#FFFDFC` | Settings、Wizard、今日详情 |
| 次级纸面 | `#F5F6F2` | Action bar、侧栏、只读状态 |
| 暖色提示面 | `#FBF5E8` | 少量推算结果与高风险说明 |
| 主文字 | `#302B26` | 标题、数值、设置名称 |
| 次文字 | `#756B61` | 状态、单位、补充信息 |
| 分割线 | `#E3E0D9` | 行与区域分隔 |
| 金币黄 | `#F2B43A` | 保存、进度、选中态 |
| 金币按下 | `#DFA029` | pressed |
| 状态绿 | `#729879` | 成功、工作中、启用 |
| 错误红 | `#AE584C` | 失败与校验错误 |

间距只使用 `4 / 8 / 12 / 16 / 24 / 32`。窗口圆角 `16`，菜单 `10`，控件 `8`，标签可使用全圆角。普通 UI 字号固定，不随视口缩放：`12 / 13 / 14 / 16 / 20 / 28 / 36`。

## 窗口合同

| Surface | 推荐尺寸 | 信息上限 | 容器策略 |
| --- | --- | --- | --- |
| Panel 折叠 | `300 × 124` | 金额、状态、细进度、下一节点 | 单一悬浮纸面，动态文本不得裁切 |
| Panel 展开 | `344 × 232` | 再增加月累计、时薪、下一节点 | 单一悬浮纸面，保留完整呼吸空间 |
| 今日详情 | `480 × 600` | 收益、进度、今日安排、月摘要 | 单层窗口 + 分割线 |
| Settings | `700 × 520` | 五页签、行式设置、固定 action bar | 单层窗口 |
| Wizard | `720 × 520` | 左侧步骤 + 当前任务 + 固定 action bar | 单层窗口 |
| Context menu | `224–240` 宽 | 当前上下文的高频操作 | 单层菜单 |

Panel 不显示 `LMM` 或 `LetsMakeMoney` 产品抬头。它是桌面上的状态部件，第一视觉层级必须直接从“今日已赚”开始。

## 控件合同

- Button：高 `36–38`，主按钮金币黄，次按钮纸面描边，pressed 为 `scale(.97)`。
- Input / Select：高 `34–36`，白纸背景，极轻边框，focus 为金币黄 2px 外环。
- Switch：`40 × 22`，无额外输入框；off 暖灰，on 状态绿或金币黄。
- Slider：轨道 `5`，thumb `15`，数值列固定宽度。
- Scrollbar：宽 `5`，透明轨道，低透明暖灰 thumb。
- 行：高 `44–48`，左侧标签，右侧控件；同一 section 通过分割线连接。

## 图标语言

- 功能图标统一采用 Lucide/Feather 风格线性图标，`18px`、`1.75px` stroke、圆角端点。
- 关闭、返回、设置等熟悉动作可使用 icon-only，但命中区不得小于 `32px`。
- 保存、恢复默认、重新配置、退出等命令保留文字，不用难猜的图形替代。
- Oil Icon 仅保留给 `40px+` 的状态/空状态入口，本轮不生成小尺寸位图图标，避免缩放后发虚。

## 动效

- hover / focus：`150ms`。
- 面板展开、页签切换：`180–220ms`，只动画 `transform / opacity / clip-path`。
- 不使用 bounce；退出比进入快。
- 必须支持 `prefers-reduced-motion`。

## 评审门禁

1. 3 秒内能判断窗口的主要任务和当前状态。
2. Settings/Wizard 不再出现第二层大框或独立内容卡。
3. Panel 折叠和展开的宽度、圆角、文字基线一致。
4. 100%、125%、150% DPI 下文字不裁切，按钮和控件不变形。
5. 所有操作具备 normal、hover、focus、pressed、disabled、success、error 状态。
6. 原型通过后再按 `window-quality-selectors.md` 映射到 Godot，不从生成方向图直接抄像素。
