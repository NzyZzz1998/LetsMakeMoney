交互说明

Panel 状态点击“折叠 / 展开 / 对照”查看常驻态与详情态的信息层级，验证折叠态只保留金额和短状态。
工作状态点击“工作中 / 休息中 / 待配置”切换金额、状态和进度条，用来确认不同业务状态的文案和视觉权重。
设置反馈点击“已保存 / 无变化 / 需重显 / 保存失败”确认设置窗口内反馈，不依赖外部弹窗。
设置导航点击左侧 Salary / Pet / Display / Panel / General，验证 Win11 式分类导航和右侧卡片内容。
右键菜单点击“窗口模式子菜单 / 选择宠物子菜单”，验证二级入口替代平铺快捷项。
图标检查 16-256 尺寸预览，确认托盘小图标需要单独优化而不是简单缩小大图。
Panel 是薪资便签折叠态只放金额和短状态，展开态才显示完整明细。
设置像系统工具保留五个分类，参考 Windows 11 设置窗口，不做宠物装饰化。
右键菜单分层窗口模式和选择宠物进入二级菜单，一级菜单只保留清晰命令。
主题系统后置v0.4 只统一默认体验，不提供 Theme 入口或外观切换。
