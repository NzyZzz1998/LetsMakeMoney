## 1. 本次范围

- 月薪按当月实际工作日拆分为日薪，不再按自然日比例估算本月累计。
- 双休仅计算周一至周五；单休额外计算每个周六。
- 大小周根据“本周是大周/小周”的锚点交替计算周六是否工作。
- 默认作息为 `08:00-18:00`，午休为 `12:00-14:00`，有效工作时间为 8 小时。
- 午休期间今日收入与工作进度暂停增长，午休结束后继续累计。
- 配置版本升级为 v4；旧 v3 配置保留原上下班时间，并根据旧有效工时推断午休时长。
- Settings、Wizard 和 Main 的薪资刷新范围接入新增配置字段。

本次不包含法定节假日数据集、加班收入、跨设备同步或新的发布包。

## 2. 计算口径

```text
当月日薪 = 月薪 / 当月实际工作日数
时薪 = 当月日薪 / 每日有效工作小时数
今日收入 = 当月日薪 × 今日有效工作进度
本月累计 = 已完成工作日 × 当月日薪 + 今日收入
```

午休区间不计入有效工作时间。午休开始与结束相同表示不设置午休；午休倒序、超出工作区间或上下班倒序均视为无效作息。

## 3. 配置字段

| 字段 | 说明 |
|---|---|
| `config_version` | v0.8 使用版本 4 |
| `rest_mode` | `double`、`single` 或 `alternating` |
| `work_start_time` / `work_end_time` | 上下班时间 |
| `lunch_start_time` / `lunch_end_time` | 午休时间；相同表示无午休 |
| `work_hours_per_day` | 根据工作与午休区间计算的有效小时数 |
| `alternating_anchor_date` | 大小周锚点所在周的周一日期 |
| `alternating_anchor_week_type` | 锚点周是 `big` 或 `small` |
