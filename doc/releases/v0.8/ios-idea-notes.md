# LetsMakeMoney Apple 产品线迁移说明

该想法最初在 Windows v0.8 工程治理之后提出，但已经确认作为独立 Apple 产品线推进，不再属于 Windows v0.8 范围。

当前事实源：

- PRD：`doc/releases/ios-v0.1/prd.md`
- 需求追踪：`doc/releases/ios-v0.1/traceability.md`
- 正式原型：`doc/prototypes/ios-v0.1/index.html`

已确认边界：

- Apple 产品线从 `ios-v0.1-beta` 开始。
- 未来使用独立长期分支 `ios-main`，不与 Windows `main` 混合开发。
- iOS v0.1 不实现 Windows 实时同步、配置口令、桌宠或加班统计。
- 本文件仅保留历史索引，不再作为需求事实源。
