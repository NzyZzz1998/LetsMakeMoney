# LetsMakeMoney v0.8 Beta 发布说明

**状态**：最终验收通过，可发布 Windows x86_64 便携 Zip

## 主要变化

- 工资按当月实际工作日计算，不再使用固定四周近似值。
- 支持单休、双休和大小周；大小周以锚点周持续推算并跨重启保存。
- 增加午休开始、结束与时长配置，工作进度和今日已赚会扣除午休时间。
- Settings 与首次向导同步展示有效工作时长，并修正非整刻时长的显示精度。
- 配置格式升级到 v4；旧 v3 配置会自动迁移，缺失字段使用安全默认值。
- 保留 v0.7 的 Panel、桌宠、菜单、托盘、点击穿透和窗口策略。

## 发布边界

- 当前仅支持 Windows x86_64 Beta 便携包。
- 宠物动作大改不属于 v0.8；相关内容只进入下一版本 Review。
- 未签名安装器不属于本候选包，也不得作为公开 Release 附件。
- 多显示器、干净 Windows 用户/VM、Authenticode/SmartScreen 和真实登录后的开机自启沿用既有“暂不验证”口径，不得写成通过。
- Windows 版与 iOS 原型/实现仍有体验差距，已记录为后续优化候选，不阻塞本次 Beta。

## 升级与回滚

- 升级前退出 LetsMakeMoney，并备份 `%APPDATA%\LetsMakeMoney\config.json`。
- v0.8 首次启动会迁移旧配置；迁移失败时应保留旧配置并记录可读日志。
- 如需回滚，退出 v0.8，恢复配置备份，再启动 v0.7 Beta 便携包。
