# LetsMakeMoney v0.8 Beta 人工验证

**状态**：等待使用新候选 Zip 复验

## 验收准备

1. 结束所有 `LetsMakeMoney.exe` 进程。
2. 备份 `%APPDATA%\LetsMakeMoney\config.json` 和 `debug.log`。
3. 将 `releases\v0.8\LetsMakeMoney-v0.8-beta-windows-x86_64.zip` 解压到新的独立目录。
4. 只运行新解压目录中的 `LetsMakeMoney.exe`。
5. 记录 Zip、EXE、DLL 的 SHA256 与启动时间。
