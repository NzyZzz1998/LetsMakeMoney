# v0.7 公开准备状态

**结论**：A0/A1/A2/A3 与远端历史清洗已经关闭仓库公开的安全、许可与隐私门禁。GitHub 公共 API 已确认仓库为 Public；后续工程、分发、社区治理和 v0.7 发布验收继续推进，不能据此宣称 v0.7 已发布。

## A0 已冻结规则

1. 当前 HEAD、脏工作区、v0.6 验收产物、v0.7 开发候选和未来公开候选必须分别记录。
2. `.gitignore` 只防止新增本地文件误提交，不能处理已跟踪文件或 Git 历史。
3. 当前树检查器默认只读、失败返回非零、只报告路径/规则/风险，不输出秘密值。
4. 完整历史、删除文件历史、作者信息和历史资产由 V07-A3 审计。
5. 任何真实凭据或无权公开资产未关闭前，仓库保持私有。
