# LetsMakeMoney 主题系统路线规划

**状态**：规划完成，v0.7 不实现

## 目标与边界

未来主题只替换设计 token 和已授权资源，不改变 Settings、Wizard、Panel、菜单的业务流程。当前暖色金币小票风是唯一正式主题与回退主题。

## 候选结构

- 颜色：纸面、文字、边框、金币、橘猫、成功、警告、错误。
- 字体：字号层级、字重和中英文回退，不捆绑来源不明字体。
- 尺寸：圆角、间距、控件高度、阴影和动画时长。
- 资源：图标、Panel 装饰和宠物搭配资源必须有独立许可记录。
- 可访问性：普通文字对比度、焦点态、禁用态和系统缩放均需验收。

主题包必须带 schema 版本、名称、作者、许可、兼容的应用版本和完整性摘要。加载失败时回退暖色主题，不允许破坏配置或阻止应用启动。

## 后续门禁

进入后续 `/idea` 前需要先证明：当前 Warm Control 已稳定、主题切换有真实用户价值、素材许可可持续。v0.7 不新增主题 API、配置字段、切换入口或第三套视觉语言。
