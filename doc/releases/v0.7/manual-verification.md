## 验收对象

仅使用 `releases/v0.7/` 中新解压的候选包和当次生成的安装器，不使用 `build/` 旧产物替代。记录 Zip、EXE、DLL、安装器 SHA256、分支和 HEAD。

## 必须人工补证

1. **通过**：项目所有者已完成人工补证。Windows 通知区真实左键可隐藏/恢复；普通模式恢复后任务栏入口存在，纯桌宠恢复后任务栏入口不存在。
2. **部分通过**：100%、125%、150%、200% DPI 下的 Panel、菜单、Settings、Wizard 均已实机通过；当前设备只有一个显示器，多显示器跨屏、记忆与断开回落暂不验证。
3. Settings 更新页：稳定/测试通道、检查中、无更新、发现更新、离线、失败、取消与重试反馈。
4. **通过**：GUI 取消、正常安装、覆盖/修复、默认保留数据卸载、显式勾选删除数据及不可恢复二次确认均已通过；验收后原用户配置已恢复。
5. **暂不验证、非版本阻塞**：安装器为 `NotSigned`，真实 Authenticode 与 SmartScreen 等待 SignPath 审核。未签名安装器不得作为 Release 附件，但不阻塞便携 Zip 和 v0.7 收口。
6. 更新：下载完成后再次确认；取消不退出；安装前生成配置备份；安装版与便携版共享 APPDATA 提示可见；失败可返回 GitHub Release。
7. 真实 Windows 登录后的开机自启继续单独记录，自动脚本不能替代真实登录证据。

每项只使用：通过、部分通过、失败、待验证、暂不验证。截图和日志证据放在本地验收目录，不提交隐私内容。
