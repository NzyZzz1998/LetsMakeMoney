# LetsMakeMoney v0.7 Beta 人工验证

**状态**：待执行

## 验收对象

仅使用 `releases/v0.7/` 中新解压的候选包和当次生成的安装器，不使用 `build/` 旧产物替代。记录 Zip、EXE、DLL、安装器 SHA256、分支和 HEAD。

## 必须人工补证

1. Windows 通知区真实左键隐藏/恢复；普通模式恢复后任务栏入口存在，纯桌宠恢复后任务栏入口不存在。
2. 100%、125%、150%、200% DPI 与至少双显示器：桌宠、Panel、菜单、Settings、Wizard 不裁切，拖拽和穿透区域不偏移。
3. Settings 更新页：稳定/测试通道、检查中、无更新、发现更新、离线、失败、取消与重试反馈。
4. 安装器：安装、取消、覆盖、修复、卸载保留数据、显式删除数据和安装日志。
5. Authenticode 与 SmartScreen：EXE、DLL、安装器发布者一致且签名有效；没有真实证书时记录为安装器发布阻塞，不得勾选通过。
6. 更新：下载完成后再次确认；取消不退出；安装前生成配置备份；安装版与便携版共享 APPDATA 提示可见；失败可返回 GitHub Release。
7. 真实 Windows 登录后的开机自启继续单独记录，自动脚本不能替代真实登录证据。

每项只使用：通过、部分通过、失败、待验证。截图和日志证据放在本地验收目录，不提交隐私内容。
