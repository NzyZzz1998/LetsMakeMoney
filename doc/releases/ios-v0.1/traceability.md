## 决策到需求映射

| 已确认决策 | PRD 需求 | 原型视图 | 主要验收 |
| --- | --- | --- | --- |
| iOS 从 v0.1 独立开始 | 全局范围 | 原型首页 | 分支、版本与文档不混用 |
| App + Widget + Live Activity + Watch | FR-004、FR-006～FR-009 | iPhone、iPad、系统入口、Watch | 四类产品形态全部闭环 |
| 真实月工作日计算 | FR-001、FR-002 | 今日、日历 | 跨月份测试向量一致 |
| 午休暂停收入 | FR-001、FR-002、FR-007 | 今日、Live Activity | 午休期间金额不增长 |
| 2025-2027 离线节假日 | FR-002、FR-005 | 日历 | 越界提示与回退正确 |
| 双休、单休、大小周 | FR-002、FR-005 | 引导、日历 | 锚点和规则优先级正确 |
| 特殊日期手动覆盖 | FR-005 | 日历 | 保存、删除和取消正确 |
| 拒绝通知仍可手动启动 | FR-007、FR-010 | 系统入口 | 权限拒绝不禁用手动入口 |
| 午休隐藏金额、显示复工倒计时 | FR-007 | Live Activity | 午休状态文案和数据正确 |
| 下班自动完成 | FR-007 | Live Activity | 完成态短暂显示并结束 |
| Watch 默认剩余时间，可切换指标 | FR-008、FR-009 | Watch | 在线、离线与切换正确 |
| iPhone 是 Watch 事实源 | FR-008、FR-011 | Watch | 离线只读且不伪造成功 |
| iPad 横屏 B+C 组合 | FR-004 | iPad | 侧栏、双栏和今日安排保留 |
| 简体中文但预留本地化 | FR-013 | 全部 | 文案资源化、无乱码 |
| 浅色与深色完整适配 | FR-012 | 全部 | 两种外观可读可用 |
| 配置口令只定义 schema | FR-014 | 无操作入口 | 无导入导出 UI，契约可测试 |
| 加班延后到 v0.2 | 非目标与延后项 | 无 | v0.1 不出现加班入口或字段 |
