# LetsMakeMoney Windows v0.9 Beta 发布说明

**状态**：最终验收通过。`V09-BUG-002` 至 `V09-BUG-008` 均已关闭，当前没有发布阻塞；实际发布状态与下载附件以 [GitHub Release `v0.9-beta`](https://github.com/NzyZzz1998/LetsMakeMoney/releases/tag/v0.9-beta) 为准。

**便携 Zip 校验**：以发布附件同目录的 `SHA256SUMS.txt` 为准。

## 主要变化

- 统一单双休、大小周、8 小时工作制、午休、节假日和调休日历的工资与进度口径。
- 重塑首次向导和 Settings，二者共享配置草稿、推算、校验与保存事务。
- 重组 Panel，并新增单实例“今日详情”窗口和安全的显示器位置回落。
- 统一 Popup、菜单、Settings、Wizard 打开期间的点击穿透保护。
- 引入通用宠物包合同、校验、导入、哈希、许可、锚点、逐帧时长和损坏回退。
- Classic Pro 保持当前稳定实现与新配置默认候选；多多成为正式可选候选并使用 S5.5 新动画载荷；旧用户选择不被自动改写。
- 动画采用事件驱动状态机；最终合同只保留状态感知单击，长按与拖动组成方向跑步交互，并配合环境动作、业务事件、指针观察和动态命中区。

## 验收结果与边界

- 当前仅支持 Windows x86_64 Beta 便携包。
- 最终候选已完成 Panel、今日详情、Settings 五页、Wizard 四步/返回/取消/关闭、菜单、Classic/多多切换与当前状态单击、Popup/Modal 点击穿透和 v0.8 核心回归。
- Windows 通知区真实鼠标左键、500ms 长按跑动和 Classic/多多完整三状态观感保留为待人工补证。
- 真实 125%/150% DPI、受控损坏包桌面表现和两小时连续运行暂不验证。
- 当前界面质感与整体前端一致性未达到项目所有者预期，作为后续版本体验债保留；v0.9 不再继续修改业务代码。
- 上述边界不得改写为通过，但按已确认验收规则不阻塞 v0.9 Beta 发布收口。

## 发布文档状态

项目所有者已选择重新打包。当前便携 Zip 内的 README 与 release notes 已同步最终验收结论；重打包只更新文档快照，EXE 与 Native DLL 仍保持已验收身份。

## 升级与回滚

- 升级前退出 LetsMakeMoney，并备份 `%APPDATA%\LetsMakeMoney\config.json`。
- v0.9 会安全迁移旧配置；旧用户的宠物选择保持不变。
- Classic 可从 Settings 一键回滚到 v0.8 橘猫；包损坏时按目标宠物、Classic、v2、v1、占位猫顺序降级。
- 如需回退版本，退出 v0.9，恢复配置备份并启动 v0.8 Beta 便携包。
