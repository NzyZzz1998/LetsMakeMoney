## 版本定位

v0.6 Beta 是一次稳定性、可诊断性和边缘体验收敛版本。它没有扩展新的产品方向，而是让现有桌宠、偏好设置、首次配置、托盘和配置保存链路更可靠，也让问题更容易定位和恢复。

## 主要改进

- Settings 与 Wizard 延续统一的暖色紧凑控件体系，保存、取消、关闭和失败反馈更一致。
- 托盘隐藏与恢复、纯桌宠任务栏策略和窗口找回路径完成稳定化与真实桌面验收。
- 右键菜单、Panel、小猫、Settings 和 Wizard 完成有限且有证据约束的体验精修。
- Popup、Settings 和 Wizard 打开期间会保护点击穿透边界，关闭后恢复桌宠交互。
- 配置保存改为安全临时写入与替换；失败时保留输入和旧配置，不显示虚假成功。
- 配置损坏时保留时间戳备份并进入安全恢复流程。
- Settings 通用页提供打开应用数据目录和复制脱敏诊断摘要，方便定位问题且不暴露薪资、坐标或用户目录。
- 日志按语义等级记录，达到 2 MB 后只保留一个轮换备份；普通模式不再持续生成交互截图。
- 验证脚本统一检查退出码和阻塞错误，降低“输出看似通过、实际失败”的风险。

## Beta 范围

本版本不包含主题系统、安装器、自动更新、多平台支持、更多宠物、动画素材大修、宠物市场或 ComfyUI 产品化。

## 已知限制

- **真实 Windows 登录后的开机自启暂不验证，不能视为已经通过。**
- 当前自动验证只证明注册表命令格式、启停事务和失败补偿逻辑，不能证明注销、重启或重新登录后一定自动启动。
- 该功能默认关闭，不影响手动启动、桌宠主流程、配置安全或托盘找回，因此不阻塞本次 Beta；发布后继续观察真实登录环境反馈。
- 已验收 Zip 内的 README 与发布说明是候选包生成时快照，仍保留“待验收”字样；当前状态以本说明、`doc/current.md` 和 `verification.md` 为准。

## 升级与回滚

1. 保留 v0.5 Beta 上一稳定发布包，不要立即删除。
2. 升级前退出 LetsMakeMoney，并备份 `%APPDATA%\LetsMakeMoney\config.json`。
3. 解压 v0.6 Beta 到独立目录后启动，不要覆盖仍在运行的旧版本。
4. 如需回滚，先退出 v0.6，恢复备份的 `config.json`，再启动 v0.5 Beta。
5. `debug.log.1` 和 `config.invalid.*.json` 不影响旧版本读取；确认问题后可自行归档。
