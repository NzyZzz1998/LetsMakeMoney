# LetsMakeMoney v0.4 Beta 发布说明

v0.4 Beta 是大型体验优化版本，重点打磨桌宠窗口可用性、交互手感、设置保存反馈、点击穿透可观测性和正式 Zip 分发体验。

## 主要变化

- 小猫交互优先级更清晰：右键菜单、拖拽、长按、双击、单击、hover 和自动状态不再互相抢状态。
- Debug 模式增加命中区调试能力，可用于检查 Pet core、Pet context 和 Panel 区域。
- Panel 边缘定位更稳，靠右、靠底和角落场景会优先选择更可见的位置。
- 设置页重新整理：缩放移动到 Display 页，Display 页集中管理窗口体验，General 页集中管理应用级设置。
- 设置保存增加差异检测和轻量反馈，无变化时不会重复写配置或注册表。
- 恢复默认显示设置前会先确认，并明确不清空薪资、工时、角色和 Panel 项。
- 日志分层：普通模式减少高频日志，`debug_mode=true` 时才输出命中区和 native passthrough 细节。
- 新增正式 Zip beta 包脚本，发布包包含 exe、native dll、README、release notes、manifest 和 checksums。

## 验证方式

发布前建议运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify_v04.ps1
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify_v03.ps1
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify_m4.ps1
```

打包：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\package_v04.ps1
```

## 已知边界

- 橘猫 v2 动画素材目录、规格和提示词已准备，但正式替换仍取决于后续素材质量筛选。
- 多缩放观感、真实桌面长时间体验和完整手动验证仍需要人工复测。
- 自动发布验证已覆盖 manifest/checksum、发布包短烟测和 60 秒 release 稳定性烟测。
- v0.4 不包含主题系统、安装器、自动更新和多平台正式支持。
