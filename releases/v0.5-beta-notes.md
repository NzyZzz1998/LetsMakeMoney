# LetsMakeMoney v0.5 Beta Release Notes

v0.5 Beta is a preference and desktop-pet edge-experience convergence release. It does not add new product directions; it focuses on making Settings, Wizard, tray restore, click passthrough protection, and verification easier to maintain and validate.

## Release Status

Final acceptance result on 2026-07-09: not publishable yet.

The implementation, package, and automated/package verification are complete, but release sign-off remains blocked by one real Windows tray manual retest: in pure-pet mode, tray left-click hide/show must restore the desktop pet without exposing a taskbar entry. Do not publish or tag this beta until that manual evidence is recorded.

## Highlights

- Added a shared Warm Control helper for Settings and Wizard.
- Migrated Settings controls to the shared compact warm style system.
- Migrated Wizard controls to the same shared style system, reducing visual drift from Settings.
- Preserved existing configuration fields and save semantics; v0.5 does not introduce a new config model.
- Added stable log events for tray restore, pure pet mode application/fallback, and passthrough suspend/resume.
- Re-applies window policy after tray restore so pure pet and passthrough behavior can recover consistently.
- Added v0.5 verification entry points and current-version documentation.
- Added v0.5 release checklist and manual verification document.

## Known Boundaries

- v0.5 does not include a theme system.
- v0.5 does not include an installer, auto update, multi-platform release, more pets, or ComfyUI productization.
- Settings/Wizard visual acceptance has passed targeted screenshot review against the v0.5 prototype direction.
- Tray restore and pure-pet recovery remain the release blocker until final real Windows tray evidence passes.
