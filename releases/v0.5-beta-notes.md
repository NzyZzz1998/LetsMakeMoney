# LetsMakeMoney v0.5 Beta 发布说明

**版本**：v0.5 Beta  
**平台**：Windows x86_64  
**状态**：通过 / 可发布  
**发布日期**：待打 tag / GitHub Release

## 这版做了什么

v0.5 Beta 是一次偏好设置与桌宠边缘体验收敛版本。重点不是增加新功能，而是让已有核心路径更稳定、更一致、更容易验收。

## 主要改进

- Settings 和 Wizard 接入共享 Warm Control 控件系统，减少两个界面之间的视觉和交互割裂。
- Settings 保存反馈补齐：
  - 保存成功。
  - 无变化保存。
  - 保存失败。
- Wizard 语义日志补齐：
  - 打开。
  - 步骤切换。
  - 完成。
  - 取消 / 关闭。
- 托盘左键隐藏 / 恢复路径补证完成。
- 纯桌宠模式恢复后保持无任务栏入口。
- 非纯桌宠模式恢复后保留任务栏入口。
- Settings / Wizard 打开期间点击穿透保护未回归。
- v0.5 发布包、manifest、checksum 和包验证脚本完成。

## 验收结论

v0.5 Beta 发布前补证验收已通过。

关键证据：

- 实际运行发布包 exe 通过。
- `verify_v05_package.ps1` 通过。
- Settings 保存成功 / 无变化 / 失败均有 UI 反馈和 `debug.log` 事件。
- Wizard 下一步 / 上一步 / 取消 / 完成均有 `debug.log` 事件。
- `pure_pet_mode=true` 恢复后主窗口为 `AppWindow=false / ToolWindow=true`。
- `pure_pet_mode=false` 恢复后主窗口为 `AppWindow=true / ToolWindow=false`。

## 已知说明

- Computer Use 无法稳定直接点击 Windows 通知区托盘图标。本轮托盘验收采用真实发布包 exe、native 托盘消息同路径、Win32 窗口样式检查和桌面截图补证。
- 点击穿透保护的 suspended/resumed 事件目前是 debug 级日志；默认 `debug_mode=false` 时不会每次写入，后续可在 v0.6 优化日志口径。
- `verify_v05.ps1` 返回通过，但 Godot headless 仍可能输出 parser 文本；发布包烟测和真实 exe 验收均通过。

## 不包含

- 主题系统。
- 安装器。
- 自动更新。
- 多平台。
- 更多宠物。
- ComfyUI 正式产品化。

## 回滚方式

如果 v0.5 Beta 发布后出现问题，可以回退到上一版 v0.4 Beta 发布包，并保留 `%APPDATA%\LetsMakeMoney\config.json` 作为用户配置备份。
