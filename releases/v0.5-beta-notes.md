# LetsMakeMoney v0.5 Beta 发布说明

v0.5 Beta 是“偏好设置与桌宠边缘体验收敛版”。本版本不新增产品方向，重点让 Settings、Wizard、托盘恢复、点击穿透保护和验证链路更容易维护、验证和回归。

## 发布状态

2026-07-09 最终验收结论：**暂不可发布**。

当前实现、发布包、自动验证和包验证已经完成，但发布签核仍被一个真实 Windows 托盘人工复测项阻塞：纯桌宠模式下，左键托盘图标隐藏/显示后，必须恢复桌宠且不能暴露任务栏入口。在该人工证据通过前，不应发布或打 v0.5 Beta 标签。

## 主要变化

- 新增 Settings / Wizard 共享暖色控件 helper。
- Settings 控件迁移到共享的紧凑暖色组件体系。
- Wizard 控件迁移到同一套组件体系，减少与 Settings 的视觉漂移。
- 保留既有配置字段和保存语义，v0.5 不引入新的配置模型。
- 增加托盘恢复、纯桌宠模式应用/降级、点击穿透暂停/恢复等稳定日志事件。
- 托盘恢复后重新应用窗口策略，使纯桌宠和点击穿透行为更容易恢复到预期状态。
- 新增 v0.5 验证入口和当前版本文档。
- 新增 v0.5 发布检查清单和手动验证文档。

## 验证命令

验收前在项目根目录运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify_v05.ps1
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify_v04.ps1
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify_m4.ps1
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify_m5.ps1
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\check_docs_status.ps1
```

打包与包验证：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\package_v05.ps1
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify_v05_package.ps1
```

## 已知边界

- v0.5 不包含主题系统。
- v0.5 不包含安装器、自动更新、多平台发布、更多宠物或 ComfyUI 产品化。
- Settings / Wizard 视觉验收已通过针对 v0.5 原型方向的定向截图复核。
- 托盘恢复和纯桌宠恢复仍是发布阻塞项，直到最终真实 Windows 托盘证据通过。
