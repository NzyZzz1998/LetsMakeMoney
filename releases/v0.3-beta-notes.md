# LetsMakeMoney v0.3 Beta 发布说明

v0.3 Beta 是“桌宠原生能力修复版”。这一版把 v0.2 暂缓的 Windows 桌宠原生能力重新接回主线：真系统托盘、透明无边框窗口、透明空白区域点击穿透、关闭隐藏到托盘、纯桌宠模式门禁和导出验证链路已经完成。

## 发布状态

- 版本：`v0.3 Beta`
- 发布 tag：`v0.3beta`
- 平台：Windows x86_64
- Godot：4.7 stable
- 本地导出产物：`build\LetsMakeMoney.exe`
- native 运行库：`build\letsmakemoney_native.dll`
- 配置路径：`%APPDATA%\LetsMakeMoney\config.json`

## 本版重点

- 应用可作为透明无边框桌宠启动。
- 小猫、Panel、右键菜单、设置窗口保持可交互。
- 小猫和 Panel 外的透明区域尽量点击穿透到桌面或下层窗口。
- Windows 右下角系统托盘提供显示/隐藏、设置、关于和退出入口。
- 点击关闭按钮时，在托盘健康状态下隐藏到托盘。
- 纯桌宠模式仍以可找回为前提，默认关闭，托盘或 native health 不满足时不会保存假成功。
- 设置中的双休/单休、窗口模式改为下拉选择，减少歧义。

## 分发注意

发布或手动复制 v0.3 时，`LetsMakeMoney.exe` 和 `letsmakemoney_native.dll` 必须在同一目录。缺少 native DLL 时，真托盘、透明窗口、点击穿透和纯桌宠模式无法按 v0.3 预期工作。

## 已知边界

- 橘猫动画仍是基础可用状态，动画质量、帧数、状态过渡和更多反馈动作进入 v0.4 Beta。
- 点击穿透目前基于交互矩形和 native hit-test，不是像素级命中。v0.4 会继续增加调试视图和边界校准。
- 设置体验已修复阻塞问题，但信息架构、保存反馈和错误提示仍会在 v0.4 继续优化。
- macOS / Linux 原生能力不包含在 v0.3 范围内。

## v0.4 方向

v0.4 Beta 将作为大型体验优化版本推进，重点包括橘猫动画打磨、交互手感、窗口/点击穿透边界、Panel 展开定位、设置体验、性能稳定性和发布包规范。
