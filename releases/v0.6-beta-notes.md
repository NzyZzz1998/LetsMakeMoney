## 主要改进

- 统一 Settings 与 Wizard 的暖色紧凑控件及保存、取消、关闭和失败反馈。
- 稳定托盘隐藏/恢复、纯桌宠任务栏策略和窗口找回路径。
- 优化 Panel、小猫、右键菜单、Settings 和 Wizard 的一致性与边缘体验。
- 保护 Popup、Settings 和 Wizard 打开期间的点击穿透边界，关闭后自动恢复。
- 增强配置安全写入、失败回滚、损坏配置备份和恢复能力。
- 新增打开应用数据目录和复制脱敏诊断摘要。
- 增加日志等级与 2 MB 单备份轮换，普通模式不再持续生成交互截图。
- 统一验证脚本的可信退出和阻塞错误判定。

## 已知限制

- **真实 Windows 登录后的开机自启暂不验证，不能视为已经通过。**
- 自动检查只能证明注册表命令格式、启停事务和失败补偿，不能证明真实登录后一定启动。
- 该能力默认关闭，不影响手动启动和桌宠核心路径，因此不阻塞本次 Beta；发布后继续观察真实登录环境反馈。
- Zip 内 README 与发布说明是候选包生成时快照，仍保留“待验收”字样；本页和仓库当前版本文档为最终口径。

## 不包含

- 主题系统、安装器、自动更新、多平台或更多宠物。
- 动画素材大修、宠物市场或 ComfyUI 产品化。
- 复杂诊断中心、诊断包上传或远程收集。

## 升级与回滚

1. 保留 v0.5 Beta 上一稳定包。
2. 升级前退出应用，并备份 `%APPDATA%\LetsMakeMoney\config.json`。
3. 将 v0.6 解压到独立目录运行。
4. 如需回滚，退出 v0.6，恢复配置备份，再启动 v0.5 Beta。
