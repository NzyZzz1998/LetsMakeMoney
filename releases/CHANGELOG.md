# 更新日志

## v0.4 Beta - 大型体验优化版（2026-07-04）

### 新增

- 新增 v0.4 橘猫动画规格、素材提示词执行集、orange_v2 staging 目录和 asset manifest。
- 新增 Debug 命中区可视化能力，可显示 Pet core、Pet context 和 Panel 命中区域。
- 新增 v0.4 自动验证脚本和发布包烟测脚本。
- 新增正式 Zip beta 打包脚本，发布包包含 exe、native dll、README、发布说明、manifest 和校验和。

### 变更

- 小猫交互优先级调整为右键菜单、拖拽、长按、双击、单击、hover、自动状态，减少误触和状态卡住。
- 单击/双击反馈改为基础状态延伸动作模型，避免把 clicked_single / clicked_double 当作独立基础状态。
- 长按阈值校准为约 0.5 秒，拖拽开始后不再触发长按语义。
- Panel 边缘定位改为候选位置 + 溢出评分，靠右、靠底和右下角更容易保持可见。
- 设置页信息架构调整：缩放移动到 Display 页，Display 页集中管理窗口体验，General 页集中管理应用级设置。
- 设置保存增加差异检测、无变化提示、保存成功/失败反馈，并减少重复写配置和注册表。
- 普通模式减少高频 debug.log；命中区、native passthrough 和托盘轮询细节只在 debug_mode=true 时写入。

- `scripts\verify_v03.ps1` 通过。
- `scripts\verify_m4.ps1` 通过。
- `scripts\verify_v04.ps1` 通过。
- `scripts\verify_v04_package.ps1` 通过，并校验 manifest / checksums。
- `scripts\verify_v04_stability.ps1 -DurationSeconds 60` 通过。

### 已知边界

- 橘猫 v2 已作为 v0.4 beta 默认资源接入，cat_orange_v1 仍保留为稳定 fallback。
- 多缩放观感、真实桌面长时间体验和完整手动验证仍需继续复测。
- 主题系统、安装器、自动更新和多平台正式支持不包含在 v0.4 Beta 范围内。

## v0.3 Beta - 桌宠原生能力修复版（2026-07-03）

### 新增

- 新增 Windows x86_64 native bridge，用于承接 Godot 原生能力不足的桌宠系统能力。
- 新增 Win32 真系统托盘：托盘图标、tooltip、左键显示/隐藏、右键菜单、设置、关于、退出。
- 新增透明无边框桌宠窗口支持，普通用户模式下不再显示明显调试窗口背景。
- 新增透明空白区域点击穿透模型，小猫、Panel、菜单和设置窗口保持可交互。
- 新增关闭隐藏到托盘能力，仅在托盘健康时启用，避免窗口不可找回。
- 新增可找回优先的纯桌宠模式门禁，托盘/任务栏控制不可用时自动禁用。
- 新增 native 构建脚本、v0.3 自动验证脚本、导出烟测脚本和 v0.3 手动验证文档。

### 变更

- 休息模式和窗口模式改为明确的下拉选择控件，不再使用容易误解的开关表达。
- 设置窗口显示原生能力状态，说明托盘、点击穿透、纯桌宠能力是否可用。
- 设置/向导打开时临时清空点击穿透区域，关闭后恢复，避免控件不可点击。
- 小猫右键菜单命中模型调整为“普通点击小命中框 + 右键专用上下文区域”，兼顾点击穿透和右键菜单稳定性。
- v0.3 配置迁移新增原生能力字段，同时保留薪资、角色、窗口、面板、自启动等已有设置。
- 发布文档改为中文，并补齐 v0.3 当前实现状态和 v0.4 后续规划。

### 修复

- 修复导出 exe 中透明窗口、任务栏控制拿到错误句柄导致的 native read 访问违例风险。
- 修复关闭按钮没有隐藏到托盘的问题。
- 修复设置进入 Display 页后影响其他选项卡保存的问题。
- 修复设置中双休/单休、置顶/融入桌面无法稳定选择的问题。
- 修复托盘左键点击不稳定的问题。
- 修复右键菜单偶发不弹出的问题。
- 修复 debug/普通桌宠模式切换后窗口尺寸和穿透区域未及时恢复的问题。

- `doc\verification\v0.3.md` 已完成手动验证反馈修复，当前无阻塞项。

### 已知边界

- 橘猫动画细节、状态过渡和动作表现不属于 v0.3 范围，统一进入 v0.4 Beta。
- v0.3 只实现 Windows x86_64 原生能力，macOS / Linux 暂不支持真托盘、点击穿透和任务栏控制。
- 发布包仍依赖 exe 与 native DLL 同目录分发，安装器和自动升级不包含在 v0.3 范围内。
