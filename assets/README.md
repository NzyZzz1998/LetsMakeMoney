Visual files under `assets/` are not covered by the repository MIT License unless explicitly stated otherwise. Runtime cat artwork and animation frames are covered by the root [restricted assets license](../ASSETS_LICENSE.md) and registered in [the asset manifest](asset-license-manifest.json).

Godot resources, text manifests, and project-authored scripts remain under the root MIT License. Third-party content must retain its original license and be recorded before publication.

Do not add visual files without updating the manifest. v0.7 does not accept external asset-file contributions.
