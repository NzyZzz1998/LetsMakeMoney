## 2. 允许行为

1. 查看公开仓库中的受限素材并运行官方二进制。
2. 为个人、非商业的评估、调试和贡献代码而克隆、编译和运行完整仓库。
3. 为向官方仓库提交代码、文档、UI 说明或 native 代码而保留未修改素材的必要仓库 fork。
4. 在 Issue 或 Pull Request 中使用必要的产品截图说明本项目问题，但不得将其中视觉作为独立素材提供。
